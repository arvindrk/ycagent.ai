# Project Documentation

This directory contains project documentation organized by category.

## Structure

### `/architecture/`

Technical architecture and design decisions:

- `ARCHITECTURE.md` - Core system architecture overview
- `SEARCH_ARCHITECTURE.md` - Search system design and implementation
- `ARCHITECTURE_IMPROVEMENTS.md` - Planned improvements and refactoring
- `SIMPLIFIED_SEARCH.md` - Simplified search approach documentation

### `/features/`

Feature-specific documentation:

- `RESEARCH_ORCHESTRATOR.md` - Research orchestration system documentation
- `SEARCH_UI_IMPLEMENTATION.md` - Search UI implementation details

## Root Documentation

Documentation at the project root:

- `README.md` - Project overview and getting started
- `README_DB_SETUP.md` - Database setup instructions
- `AGENTS.md` - AI agent skills and capabilities reference

## Contributing

When adding new documentation:

- Place architecture docs in `/architecture/`
- Place feature docs in `/features/`
- Keep setup/getting started docs at project root
